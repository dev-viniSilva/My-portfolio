Game World

Gameworld é um projeto interativo feito com JavaScript moderno, que apresenta um sistema de seleção de personagens de fantasia usando programação orientada a objetos e manipulação do DOM. Ele também inclui integração com backend usando Node.js e Nodemailer, permitindo que os usuários enviem e-mails por meio de um formulário — unindo interatividade front-end com funcionalidade back-end em uma experiência full-stack.
